# NQueenHillClimbing
## Solving N Queens problem using Hill Climbing

### N-queen

N-queen problem revolves around placing N number of queens on a NxN chessboard such that no two queens can attack each other. Problem Formulation: Goal: N queens are on the board, with no queens attacking each other States: all arrangements of 0-N queens on the board Initial state: 0 queens on the board Successor function: each successor is obtained by adding one queen in an empty square
